LICON

Automate LinkedIn connection requests from any company page or search results.

How it works
1

Open a LinkedIn page

Go to a company's People tab or use LinkedIn search with filters.

2

Configure settings

Set profile limits, delay range, and behavior options.

3

Hit Start

LICON scrolls through profiles and sends connection requests automatically.
